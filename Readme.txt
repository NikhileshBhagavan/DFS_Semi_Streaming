Commands to build the system :
1. make clean
2. make real -> TO RUN ON REAL GRAPHS
   make random -> TO RUN ON RANDOM GRAPHS

Commands to run the system on different types of graphs:

-> TO RUN THE SYSTEM ON ALL REAL GRAPHS:
    ./runreal.sh

-> TO RUN THE SYSTEM ON RANDOM GRAPHS:
    ./dataGen.sh TYP N SP TST (Check the "dataGen.sh" file for detailed description)
